Explainable astronomy

A compact exoplanet physics chapter

The interface now treats the model layer like a textbook page: display equations, symbols, units, and assumptions are separated from the prose. These are simplified observing equations for public science and target triage, not a replacement for a full retrieval or orbital fit.

★ stellar quantity p planetary quantity ⊕ Earth-normalized unit ☉ solar-normalized unit
01

Incident stellar flux

How strongly the planet is irradiated relative to Earth.

\[ S_p = \frac{L_\star / L_\odot}{\left(a / \mathrm{AU}\right)^2} \tag{1} \]

A planet with \(S_p = 1\) receives the same bolometric flux Earth receives from the Sun. This is the quantity used for the radius-flux population map and for a first-order climate triage.

\(L_\star\)
stellar luminosity
\(a\)
semi-major axis
\(S_p\)
planet incident flux in Earth units
02

Equilibrium temperature

A zero-dimensional temperature estimate controlled by albedo.

\[ T_{\mathrm{eq}} = T_{\mathrm{eff}}\left(\frac{R_\star}{2a}\right)^{1/2} \left(1-A_B\right)^{1/4} \tag{2} \]

The albedo slider changes \(A_B\). The expression assumes global heat redistribution and no greenhouse correction, so it is a diagnostic temperature rather than a climate model.

\(T_{\mathrm{eff}}\)
stellar effective temperature
\(R_\star\)
stellar radius
\(A_B\)
Bond albedo
03

Kopparapu-style habitable-zone boundaries

Climate boundaries expressed as stellar-flux limits and then converted to orbital distance.

\[ S_{\mathrm{eff}} = S_{\mathrm{eff},\odot} + aT'_\star + bT_\star'^2 + cT_\star'^3 + dT_\star'^4 \tag{3a} \]
\[ d_{\mathrm{HZ}} = \left(\frac{L_\star / L_\odot}{S_{\mathrm{eff}}}\right)^{1/2}\,\mathrm{AU}, \qquad T'_\star = T_{\mathrm{eff}} - 5780\,\mathrm{K} \tag{3b} \]

The site uses inner and outer flux limits to classify a planet as inside, hotward of, or coldward of the optimistic and conservative habitable-zone bands.

The plotted HZ panel shows the orbit against those boundaries; it does not imply habitability without atmosphere, geophysics, stellar activity, and composition constraints.

04

Transit depth and duration

Photometric observability from the ratio of planetary and stellar radii.

\[ \delta_0 \simeq k^2, \qquad k = \frac{R_p}{R_\star} \tag{4a} \]
\[ T_{14} \simeq \frac{P}{\pi}\arcsin\!\left[ \frac{R_\star}{a}\frac{\sqrt{(1+k)^2-b^2}}{\sin i}\right] \tag{4b} \]

The Python light-curve figure adds a simple limb-darkened shape around this geometric depth. The catalog readout reports depth in parts per million.

\(k\)
radius ratio
\(b\)
impact parameter
\(i\)
orbital inclination
05

Radial-velocity semi-amplitude

The stellar reflex velocity induced by an orbiting planet.

\[ K = \left(\frac{2\pi G}{P}\right)^{1/3} \frac{M_p\sin i}{\left(M_\star + M_p\right)^{2/3}} \frac{1}{\sqrt{1-e^2}} \tag{5} \]

When inclination is missing, the app uses an edge-on approximation. The plotted curve is a Keplerian observing model from catalog parameters, not a fit to raw spectra.

Earth-unit shortcut: \(K \approx 0.08945\,\mathrm{m\,s^{-1}}\,(M_p/M_\oplus)(P/\mathrm{yr})^{-1/3}(M_\star/M_\odot)^{-2/3}\sin i/\sqrt{1-e^2}\).

06

Bulk density, gravity, and escape velocity

First-order structural diagnostics from mass and radius.

\[ \rho_p = \rho_\oplus\frac{M_p/M_\oplus}{(R_p/R_\oplus)^3}, \qquad g_p = g_\oplus\frac{M_p/M_\oplus}{(R_p/R_\oplus)^2} \tag{6a} \]
\[ v_{\mathrm{esc},p}=v_{\mathrm{esc},\oplus} \left(\frac{M_p/M_\oplus}{R_p/R_\oplus}\right)^{1/2} \tag{6b} \]

These quantities help separate likely rocky worlds, volatile-rich sub-Neptunes, and gas giants. They are most meaningful when mass and radius are both measured rather than inferred.

07

Transmission-spectroscopy signal proxy

A scale-height estimate for atmosphere follow-up prioritization.

\[ H = \frac{k_B T_{\mathrm{eq}}}{\mu m_H g_p} \tag{7a} \]
\[ \Delta_{\lambda} \sim \frac{2NHR_p}{R_\star^2} \tag{7b} \]

The atmosphere menu changes \(\mu\), the mean molecular weight. The scale-height slider changes \(N\), the number of scale heights assumed to contribute to spectral modulation.

This is intentionally a proxy: clouds, hazes, composition, stellar contamination, and instrument systematics can dominate real observations.

How exoplanets are detected

Discovery methods connected to observables

Each method measures a different physical signature, so the science lab reports the quantities that matter for follow-up rather than only the discovery label.

Transit photometry

Detects a small, periodic dimming when a planet crosses the stellar disk. The depth constrains radius; repeated events constrain period; duration and shape constrain orbital geometry.

Radial velocity

Measures Doppler shifts in stellar spectra caused by the star orbiting the system barycenter. It gives a mass-related signal and is strongest for massive planets close to low-mass stars.

Direct imaging

Separates faint planetary light from a bright host star. It favors young, hot, massive planets at wide separations where contrast and angular separation are less punishing.

Microlensing

Uses the gravitational field of a foreground system as a lens. It can reveal planets far away, but the events are usually non-repeatable and follow-up characterization is difficult.

Open data workflow

Built for reproducible public publishing

The repository includes a Python script that queries the NASA Exoplanet Archive TAP service, converts selected fields into a browser-ready JSON catalog, and keeps the static site deployable through GitHub Pages.

View repository

1. Static site

No build step is required. The browser loads index.html, styles.css, app.js, and data/exoplanets.json.

2. Science fields

The updater requests stellar mass, stellar radius, luminosity, inclination, eccentricity, insolation, density, and photometric fields when available.

3. Data refresh

Run python scripts/fetch_exoplanets.py locally or trigger the included GitHub Actions workflow to update and redeploy the JSON catalog.
