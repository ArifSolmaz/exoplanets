How exoplanets are detected

Discovery methods explained without hand-waving

Exoplanet science often infers planets from how they affect the light or motion of their host stars. The site’s method filters connect those observing techniques to real planets.

☌

Transit

A planet crosses in front of its star and blocks a small fraction of starlight. Repeated dips reveal orbital period; dip depth helps estimate radius.

↔

Radial velocity

A planet’s gravity makes its host star wobble. Doppler shifts in the stellar spectrum reveal a minimum planet mass and orbital period.

◉

Direct imaging

Telescopes suppress the host star’s glare and capture light from young, hot, wide-orbit giant planets directly.

⌁

Microlensing

Gravity from a foreground star-planet system briefly magnifies background starlight, creating a telltale lensing signal.

Exploration milestones

A compact exoplanet exploration timeline

  1. 1995

    First planet around a Sun-like star

    51 Pegasi b established hot Jupiters as a major class of close-orbiting giant planets.

  2. 2009

    Kepler changes the scale of the field

    The Kepler mission made transits the dominant engine of exoplanet population statistics.

  3. 2016

    Nearby small-world discoveries accelerate

    Proxima Centauri b and the TRAPPIST-1 system focused attention on nearby M-dwarf systems.

  4. 2018

    TESS begins an all-sky search

    TESS prioritizes bright, nearby stars, producing targets well suited for follow-up studies.

  5. 2020s

    Atmospheres become a public focal point

    Space telescopes and high-resolution ground instruments increasingly probe exoplanet atmospheres.

Open data workflow

Built for reproducible public publishing

The repository includes a Python script that queries the NASA Exoplanet Archive TAP service, normalizes selected fields, and writes the static JSON file consumed by this page.

View repository

1. Static site

No build step is required. The browser loads index.html, styles.css, app.js, and data/exoplanets.json.

2. Data refresh

Run python scripts/fetch_exoplanets.py locally or trigger the included GitHub Actions workflow to update the JSON catalog.

3. Public deployment

Deploy from the repository root or use the included Pages workflow. The project is ready for https://arifsolmaz.github.io/exoplanets/.
